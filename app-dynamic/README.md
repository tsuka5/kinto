# Kinto NEON — 超ダイナミック版

`app/` と**同じデータ・同じロジック**（栄養マッチ・相場・検索・レシピ）を、全く別のビジュアルで包んだバリアント。

## app/ との違い（見た目・演出のみ）
- **ダークネオンテーマ** … 深宇宙ネイビー地に、ティール→スカイ→バイオレット→ピンクの流れるグラデーション。
- **パーティクルネットワーク背景** … Canvasで星座のように光る粒子が漂い、近い粒子同士が線で結ばれる（ライブラリ不使用）。
- **相場ティッカー** … ヘッダー下を株価ボード風に全食材の相場が流れる。タップでその食材の詳細モーダルへ。ホバー/長押しで一時停止。
- **グラデーション文字ロゴ＋グラデーション枠ヘッダー**、ネオン発光する栄養バー・ボタン・ランキング番号（パルスリング）。
- **3D演出強化** … カードはブラーが解けながら奥から起き上がり、画面遷移は3Dフリップ、モーダルは傾きながらスライドアップ。カードのポインタ追従チルト・採用時の紙吹雪・スクロール視差は `app/` と共通。
- `prefers-reduced-motion` では全アニメーション（パーティクル含む）を無効化。

## 使い方
`app/` と同じ。`index.html` をダブルクリック、または `npx serve app-dynamic` で配信（全食品DB検索は配信時のみ）。

## メンテナンス上の注意
- ロジックの修正は `app/index.html` 側を正とし、このフォルダへは `scratchpad` の `build_neon.js` 相当の手順（スタイル差し替え＋ティッカー/パーティクルJS追加）で反映するか、該当箇所を手で同期する。
- Service Worker のキャッシュ名は `kinto-neon-v1`（`app/` と衝突しない）。localStorage のキーは共通（同一オリジンに両方デプロイした場合、記録は共有される）。
