🏷️ 店頭で相場チェック

スーパーで値札を見たら、その場で「安いかどうか」を判定。食材を選んで値段と内容量を入れるだけ。

💪 栄養コスパ

カテゴリ別に「栄養充実度 ÷ 相場」が高い順。タップで相場・栄養・代替を表示します。
地域（相場の基準）
相場をこの地域基準で表示します。現在地はIPからおおよそ自動推定（誤差・VPNで外れることあり）。手動で変更できます。
※IP判定は外部サービス（ipwho.is / ipapi.co）にIPを送ります（健康データは端末内のまま）。判定した都道府県の県庁所在市の実売価格を相場更新で使います。
📡 相場データの自動更新（e-Stat 実データ）
政府統計「小売物価統計調査」から今月の実売価格を取得して相場を更新します。設定は不要（アプリ内蔵の共有キーで動きます）。月が変わるたびに自動更新され、値下がりした食材が上の電光掲示板と「値動き」タブに反映されます。
自分の appId を使いたい場合のみ：e-Stat API で発行 → 下に貼り付け（共有キーより優先されます）。
